# Exoplanet Light Curve Analyzer

## About This Project
Inspired by Dr. Webb's research on light helping study Blazars. This project will receive data from NASA's Kepler missions via lightkurve and create light curves on plots which can be used to analyze dips.

## Key Features
* **Automated Data Retrieval:** Fetches real observational data directly from NASA's archives.
* **Time-Series Visualization:** Plots raw stellar flux over time using `matplotlib`.
* **Transit Analysis:** Highlights the periodic dips in brightness caused by exoplanets passing between the host star and the telescope.
* **Will Keep being worked on

## Technologies Used
* **Python 3**
* **Lightkurve:** For NASA database querying and Kepler data manipulation.
* **Matplotlib:** For rendering 2D data visualizations and graphs.
